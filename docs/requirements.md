# user_insight_pm v1.0 需求基线

## 项目目标

- 提供东风日产企微助手“用户洞察”静态产品原型。
- 支持通过统一演示入口查看业务页面和交互说明。
- 保持无构建步骤，可直接通过静态 HTTP 服务运行。

## 功能范围

- 密码访问入口和原型页面导航。
- 渠道效果、培育运营、用户群体洞察三大业务分页。
- 筛选、KPI、图表、排行、下钻弹窗和明细下载演示。
- Mock 数据、功能标注和交互说明文档。
- 独立工作台演示页面。

## 非本期范围

- 真实后端接口、登录鉴权和权限校验。
- 生产数据接入及数据持久化。
- 前端框架、构建工具和包管理器。
