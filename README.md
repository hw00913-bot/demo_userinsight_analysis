# 用户洞察 (User Insight) - 产品原型

## 项目简介

用户洞察是一个用于分析客服通话记录中。

## 包含页面

- `index.html` - 密码验证入口
- `interaction_docs.html` - 交互逻辑说明文档
- `main.html` - 主容器页面
- `userinsight.html` - 用户洞察看板
- `workbench.html` - 独立工作台页面
