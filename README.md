# 用户洞察 (User Insight) - 产品原型

## 项目简介

用户洞察是一个用于分析客服通话记录中常见问题、热门问题和客户抗拒点的数据看板系统。

## 包含页面

- `index.html` - 密码验证入口
- `interaction_docs.html` - 交互逻辑说明文档
- `main.html` - 主容器页面
- `userinsight.html` - 用户洞察看板
- `manual_review.html` - 人工复核任务中心
- `manual_review_detail.html` - 人工复核详情页
